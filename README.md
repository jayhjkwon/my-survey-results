## Prerequisites
- Node.js v8.0+

## Quick Start
```sh
npm install
npm run dev
```
Then visit http://localhost:3000

## Run test
```sh
npm test
```

## Run in production mode
```sh
npm start
```
Then visit http://localhost:3001

## Tech stack used for this app
- React
- Redux
- React-Router
- Styled Components
- Bootstrap
- Jest
- Enzyme
